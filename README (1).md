# 🚗 Automobile Data Cleaning & Exploratory Data Analysis

## 📌 Project Overview

This project focuses on cleaning, preprocessing, transforming, and analyzing automobile data using Python. The goal is to prepare raw automobile data for analysis and discover meaningful insights through Exploratory Data Analysis (EDA).

The project demonstrates essential data science techniques such as:
- Data Cleaning
- Handling Missing Values
- Data Wrangling
- Feature Transformation
- Data Normalization
- Data Binning
- Correlation Analysis
- Data Visualization

## 📂 Project Structure

├── Automobile_data_Cleaning.ipynb
├── Automobile_Data_Wrangling_EDA.ipynb
└── README.md

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📊 Key Features

- Data Cleaning and Preprocessing
- Missing Value Treatment
- Data Type Conversion
- Feature Engineering
- Data Normalization
- Data Visualization
- Exploratory Data Analysis (EDA)

## 📈 Insights

- Engine size strongly affects vehicle price.
- Higher horsepower generally increases price.
- Fuel consumption impacts automobile pricing.
- Data normalization improves feature comparison.
