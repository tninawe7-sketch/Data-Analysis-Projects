# ✈️ Airline Data Analysis & Visualization

## 📌 Project Overview

This project focuses on analyzing and visualizing airline/immigration data using Python, Pandas, and Matplotlib. The objective is to perform data filtering, exploratory data analysis (EDA), and create meaningful visualizations to uncover trends and patterns in the dataset.

The project demonstrates:
- Data Loading and Preprocessing
- Data Filtering and Analysis
- Exploratory Data Analysis (EDA)
- Statistical Summaries
- Data Visualization using Pandas
- Data Visualization using Matplotlib
